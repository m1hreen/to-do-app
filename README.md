# CLI To-Do App

A simple command-line To-Do application built using Python.

## Features
- Add tasks
- View tasks
- Delete tasks
- Mark tasks as complete
- Saves tasks using file handling

## Technologies Used
- Python
- File Handling
- VS Code

## How to Run
1. Install Python
2. Open terminal in project folder
3. Run:

python todoapp.py

## Project Purpose
This project was created to practice Python basics, loops, conditions, lists, and file management.
